Различные инструменты, в виде набора готового к использованию. Некоторые наборы либо позволяют совершать связанные с этим активности которые обычными руками невозможны или дают бонус к проверке. Каждый набор обладает ограниченным количеством использованием. Во время игры можно придумать различные инструменты которые могу помочь в различных активностях. 

Примеры:
- набор приключена - необходимые предметы для походов, спальный мешок, набор факелов, веревка 50 футов, бурдюк с водой на 2 дня, походная еда на 7 дней, и кусок мела
- набор путешественника - необходимые предметы для длительного путешествия. Расширенный набор приключена с запасом воды на 7 дней, еды на 21 день, и дающим бонус +1 при проверках выживания
- набор медика - необходимые инструменты для оказания первой помощи. Содержит простые скальпели, ножи и зажимы а также набор расходников в виде бинтов, трав и мазей на 4 использования
- набор врача - расширенный набор медика. Количество расходных вещей увеличено до 8 использований. При использовании дает +1 бонус при проверках медицины
- набор воровских инструментов - инструменты для взлома замков или отключения тонких механизмов ловушек. Позволяет производит проверки воровства для вскрытия замков или отключении ловушек не поднимая шума. Может использовать множество раз. И может пережить 3 провала до того как сломаются все отмычки.
- набор взломщика - расширенный набор воровских инструментов. Дает +1 при проверках  воровства для вскрытия замков или отключении ловушек и может пережить 6 провалов
- набор скалолаза - набор веревок, кольев, зажимов и прочих инструментов что-бы взбирается по вертикальным скалам. Дает +1 к проверки атлетики для подъёму или спуску по вертикальной стене и в случае провала спасает от падения. Этими бонусами могут воспользоваться каждый если не убрать веревку. Может помочь преодолеть 50 футовое расстояние. После использование нужно пополнить комплект веревок и кольев.
- набор альпиниста - расширенный набор скалолаза, Дает +2 к проверкам атлетики для подъёму или спуску по вертикальной стене. Позволяет преодолеть 100 футов по вертикали. В комплекте есть пару болтов с крюк кошкой для использовании с арбалетом чтобы перебросить веревки через горизонтальный обрыв. Также защищает от падения даже при крит. провале. 
- набор алхимика - позволяет во время отдыха за 30 минут приготавливать различные зелья. Обычный набор позволяет приготовить 3 зелья. Расширенный дает +1 к проверке ремесла для изготовления зелья и может быть использован 6 раз
- набор для ремонта - позволяет в походе быстро чинить оборудование. Обычный может быть использован 2 раза. Расширенный дает +1 к проверке ремесла для починки и может быть использован 3 раза.
- набор актера - позволяет менять внешность специалистам по гриму или подделывать различные документы. Обычный позволяет за 1 час замаскироваться или подготовить подделку 3 раза. Расширенный дает бонус +1 к проверкам выступления для маскировки и подделки документов и можно это сделать 4 раза.
- пояс для зелий - позволяет подготовить в снаряжение 3 зелья в место одного
- увеличенный пояс для зелий - позволяет подготовить в снаряжение 6 зелий в место одного
- подсумок - позволяет подготовить 2 расходника в место одного
- боевой пояс - позволяет подготовить 4 расходника в место одного
